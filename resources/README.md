### Command 2: `/inverseenchant <player> <enchant> <level>`
**What it does:** Gives an enchant directly to someone's held item

**How to use:** 
```
/inverseenchant @s Life_Steal 2
```

**What that means:**
- `@s` = yourself
- `@p` = closest player
- `@a` = all players  
- Or type a player's name!
- `Life_Steal` = the enchant name
- `2` = the level (1, 2, or 3)

**Easy Examples:**
- `/inverseenchant @s Bleeding 1` â†’ Give yourself Bleeding level 1
- `/inverseenchant @p End_Slayer 1` â†’ Give the closest player End Slayer
- `/inverseenchant @a Speed_Boost 3` â†’ Give everyone Speed Boost level 3
- `/inverseenchant PlayerName Experience 2` â†’ Give a specific player Experience level 2
